inlineprivatevirtual

Called when a new object is added into the simulation.

This allows an observer such as a tabulator to examine objects immediately when they are added to the simulation.

Parameters
objectA pointer to the object
objTypeAn integer that defines the type of object so that it can be properly cast. The values for this should be defined as either static constants are using enumeration in the subclass of JPObservableSimulation

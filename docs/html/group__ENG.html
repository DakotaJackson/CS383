void JPSimulationEngine::step ( double  sec)

Advance the simulation by sec seconds.

Advance the state of the simulation by sec seconds.

This function causes the simulation engine to perform the calculations and modification necessary to advance the simulation. For example if step(0.1) is called, all cars in the simulation will be moved to their positions 0.1 seconds from now. It also advances the simulation's internal clock.

Parameters
secThe number of seconds to advance the simulation.
